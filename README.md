## ⚙️ 개발 환경 정보 (Environment)

| 항목 | 상세 내용 |
| :--- | :--- | 
| **프레임워크** | **React 19 + Vite 7** |
| **언어** | **TypeScript 5.8** | 
| **스타일링** | **Styled-Components 6.1** | 
| **상태 관리** | **Jotai 2.15** | 
| **API 통신** | **Axios 1.12** |
| **패키지 매니저** | **npm** | 

---

## 💻 실행 방법 (Getting Started)

```bash
1. 레포지토리 복사
git clone https://github.com/YU-MIDAS-26/Frontend.git

2. 프로젝트 폴더로 이동
cd Frontend

3. 의존성 패키지 설치
npm install

4. 로컬 개발 서버 실행
npm run dev
